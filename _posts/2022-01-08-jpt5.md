##### Q2.
객체 변수 value가 100이상의 값을 가질 수 없는 프로그램


class Calculator:
    def __init__(self):
        self.value = 0
        
    def add(self, val):
        self.value += val

class MaxLimitCalculator(Calculator):
   def add(self, val):
	self.value += val
	if self.value > 100:
		self.value = 100
        
cal = MaxLimitCalculator()
cal.add(50)
cal.add(60)
print(cal.value)

add함수를 오버라이딩해서 self값이 100이 넘는 경우를 if문 처리하면 
됐을 문제.  내가 너무 꼬아서 생각을 했다...
